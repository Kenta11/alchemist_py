# 開発フロー

Alchemistはコンポーネントを組み合わせて回路システムを構築します．
コンポーネントの組み合わせ方はpublish/subscribe通信ネットワークの形式で表現します．
設計したpublish/subscribe通信ネットワークをTOMLファイルに記述することで，必要なファイルの生成や回路の合成をAlchemistが行います．
