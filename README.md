# alchemist

Alchemist is a Component-Oriented and High-Level-Synthesis (HLS) FPGA Development Environment Based-on DDS (Data Distribution Service).

[日本語版](README-jp.md)


## Usage

### Creating a new project

```
$ alchemist new
```

### Update the project

```
$ alchemist update
```

### HLS simulation

#### Unit testing

```
$ alchemist test --unit
```

#### Integrated testing

```
$ alchemist test --integration
```

### Bitstream generation
```
$ alchemist compile
```
